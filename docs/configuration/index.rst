Configuration
=============

Configuration reference for snooze-server. When using the packaged snooze, the configuration
files are located under ``/etc/snooze/server/``.

Some of these configuration are editable in the web interface with an admin account.
When edited that way, snooze-server will override the config file in ``/etc/snooze/server/``.
